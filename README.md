# 🌿 AssurancesNet.fr — Comparateur Santé Senior

Site statique HTML · Déployé sur Vercel · Domaine : assurancesnet.fr

## Structure
```
index.html          ← Page d'accueil
vercel.json         ← Config URLs propres
sitemap.xml         ← Toutes les pages pour Google
robots.txt          ← Instructions crawlers
blog/               ← Tous les articles et comparatifs
```

## Déployer
1. Pousser sur GitHub
2. Importer sur vercel.com → Deploy
3. Ajouter assurancesnet.fr dans Settings → Domains

## Ajouter un article
Créer un fichier HTML dans /blog/ puis ajouter l'URL dans vercel.json et sitemap.xml
